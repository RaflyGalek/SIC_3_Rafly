1. Sebutkan step-step dan command line saat menginisasi project untuk menggunakan git hingga
sampai push ke github
- set up konfigurasi  git. cth (git config --global user.name "Peserta SIC")
- memeriksa apakah set up berhasil (git config --list)
- membuat repository (git init proyek_IoT)
- buat file (sensor.py)
- tambahkan file ke index (git add sensor.py)
- simpan perubahan (git commit -m "")
- (git push origin master)

2. Apa perbedaan git reset dan git revert
- git reset : mengembalikan file kemudian menghapus catatan sejarah commit 
- git revert : mengembalikan file dengan tidak menghapus sejarah commit

3. Apa bedanya GIT dengan GITHUB
- GIT dapat diakses menggunakan penyimpanan local secara offline
- GITHUB menyediakan lanyanan cloud dan diakses secara online
